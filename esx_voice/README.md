# esx_voice
FXServer ESX Voice Version+


#### Description
This is a proximity voice controller script. The default control for this is `Left Shift + H` (QWERTY).

#### Installation

1) Add `start esx_voice` to your server.cfg
2) Delete cache folder in server-data
